# どきどきサイコロゲーム

ブラウザ上で遊べるシンプルなサイコロゲームです。  
最初に出たサイコロを基準値として表示し、次の目が「大きい」か「小さい」かを予想して連続で正解を重ねます。

## 遊び方

1. ブラウザで [index.html](index.html) を開くか、ローカルサーバーを立ち上げて表示します。
2. 「スタート」を押して、最初のサイコロを振ります。
3. 表示された基準値を見て、次の目が大きいか小さいかを選びます。
4. 正解すると連続成功数が増え、次のゲームに進めます。
5. 同じ目が出た場合や不正解の場合はゲームオーバーになります。
6. 3回連続で正解するとゲームクリアです。

## 実行方法

ローカルで確認する場合は、プロジェクトのルートで次のコマンドを実行してください。

```bash
python3 -m http.server 8000
```

その後、ブラウザで http://localhost:8000/ を開きます。

## ファイル構成

- [index.html](index.html)  : 画面レイアウト
- [styles.css](styles.css) : スタイルとレイアウト
- [script.js](script.js)   : ゲームロジック、アニメーション、音声演出

## 特徴

- シンプルな1ページ構成
- サイコロの回転アニメーション
- 成功・失敗時の音声演出
- スマホサイズに合わせたレイアウト
